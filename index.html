GIT: Comandos para iniciar proyecto

  1. Abrir la app de git
  2. Para configurar el nombre escribir: git config --global user.name "Gaby"
  3. Para configurar el correo escribir: git config --global user.email "maria.gabriela.acosta.v@gmail.com"
  4. Para configurar los colores de git escribir: git config --global color.ui true
  5. Para ver el listado de nuestra configuracion: git config --global --list

Para salir de alguna ventana ponemos la letra q

Se puede escribir git help o tambien git help nombre-comando para ayuda

Eliminar los Commit


  • git reset --soft: elimina los commit pero no toca el código.
  • git reset --mixed: no recomendable
  • git reset --hard: borra codigo y todo, no deja rastro de nada

Github

Para poder sincronizar el repositorio local con el remoto usamos github

  1. Para crear una copia de un proyecto online en nuestro local: git clone url_dado_git
  2. Para vincular (conectar) nuestro proyecto local con el proyecto remoto escribimos: git remote add origin url_nuestro_repo
  3. Para comprobar si se vinculo el repositorio local con el remoto: git remote -v
  4. Para desvincular git hub de nuestro local escribimos: git remote remove
  5. Para enviar nuestros cambios (commits) a github escribimos: git push origin nombre_rama ó git push -u origin master
  6. Para subir cambios del local a github (despues de almacenar en el local): git push origin master
  7. Para subir cambios de otra rama en el local al remoto: git push origin nombre-rama
  8. Para subir cambios forzados como un git commit amend escribo: git push origin nombre-rama -f
